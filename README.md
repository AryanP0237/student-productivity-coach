# Student Productivity Coach

A simple desktop assistant built in Python + Tkinter that:
- Tracks study sessions with a timer
- Logs tasks and suggested plans to `session_log.txt`
- Analyzes your mood and gives basic recommendations

## Tech Stack
- Python 3
- Tkinter (GUI)
- TextBlob (sentiment analysis)
- Git for version control
